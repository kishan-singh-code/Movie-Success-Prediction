Our Work

Scope

The focus of this project is to formulate a method of how to preprocess the data set and evaluate which attributes are the most useful, by evaluating the correlation between the attributes and the success rate of the data mining techniques. Using this method we can achieve a viable success rate when trying to predict the rating and box office revenue. The data set is going to be obtained on IMDb, Kaggle, and The Movie DB. The data set will be limited to 10,000 unique movies because of making the workflow more manageable when processing the data. The algorithms that are going to be used are mainly, KNN, Random Forest, and Social media prediction.
